# GIT
En la utilizacion de GIT tuvimos una leve confusion, donde en un momento dado que en un merge y pull request una parte del trabajo fue subido a la rama main
en vez de a la rama dev, por lo que tuvimos que rehacer el repositorio y eso implicaba que se borren los varios commits, merges y pull request que habiamos realizado.
Luego de rearmar el repositorio no hubo mas confusiones y cada uno de los integrantes pudo trabajar tranquilo y hacer bien los merges y pull request a las ramas correspondientes.
